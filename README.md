# Java Swing Text Editor

This is a simple text editor application implemented using Java Swing.

## Overview

The Java Swing Text Editor provides a basic yet functional text editing environment for users. It allows users to create, edit, and save text documents with ease.

## Features

- **Create New Documents**: Allows users to create new text documents.
- **Open Existing Documents**: Provides functionality to open existing text documents for editing.
- **Save Documents**: Enables users to save their documents to a specified file path.
- **Text Editing**: Provides standard text editing features such as copy, cut, paste, undo, and redo.

## Technologies Used

- Java Swing (for the user interface)
- Java File I/O (for reading from and writing to text files)
